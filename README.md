# CRUD PHP MongoDB

## Update
1. Composer Ready
2. OOP Ready
3. Simple Router
4. Bootstrap 4.6.0
5. DataTables 1.10.23

## Installation
```sh
git clone https://github.com/suendri/free-crud-php-mongodb.git
setting Connection app/Controller.php
set RewriteBase on .htaccess based on folder name
```
